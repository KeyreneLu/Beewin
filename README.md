# Beewin
公司运营项目，停运了

###技术选型
* 网络：RxJava+Retrofit
* 图片：Glide
* 6.0动态权限：需额外加依赖（RxJava）rxpermissions
* 推送：极光推送
* 分享：友盟分享
* 网页：腾讯X5
